# מפת כישור הזמן האינטראקטיבית (Wheel of Time Interactive Map)

יישום אינטראקטיבי מתקדם ודו-לשוני (עברית ואנגלית) המאפשר ניווט חלק, חיפוש, סינון, ומיפוי של ערי ואתרי מפתח מתוך עולם הפנטזיה של "מחזור כישור הזמן" (Wheel of Time), על גבי תמונת המפה ברזולוציה הגבוהה (`afavl8.jpeg`, בגודל 7146x5079 פיקסלים).

האפליקציה כוללת **כלי עריכה ומיפוי מובנה (Editor Mode)** המאפשר להציב אתרים על המפה בלחיצת כפתור ולשמור את המידע ישירות לתוך קובץ הנתונים בדיסק המקומי (`locations.json`).

---

## תכונות מרכזיות

1. **ניווט וזום אינטראקטיביים**: תמיכה בגלילה, מתיחה וזום חלקים במיוחד באמצעות הספרייה המובילה `Leaflet.js` על גבי מערכת קואורדינטות פשוטה מבוססת פיקסלים.
2. **עיצוב יוקרתי (Premium Dark Fantasy Theme)**: עיצוב ממשק משתמש אלגנטי, שימוש ב-Glassmorphism (שקיפויות וטשטושים), צבעי זהב עמוקים, עיטורים מותאמים ואנימציות.
3. **דו-לשוניות מלאה (Hebrew / English)**: תמיכה מלאה בהחלפת שפה בזמן אמת עבור כל הממשק, כיווניות הטקסט (RTL/LTR) ותיאורי האתרים במפה ובסרגל הצד.
4. **חיפוש וסינון מתקדמים**: סינון מהיר לפי קטגוריות (ערי בירה, ערים, אתרים) וחיפוש דינמי בזמן אמת לפי שמות ותיאורים בשתי השפות.
5. **מערכת שמירה כפולת-נתיבים**: שמירת נתונים ישירות לדיסק המקומי באמצעות שרת Express קטן, עם מנגנון גיבוי אוטומטי ל-Local Storage בדפדפן למקרה של עבודה ללא שרת.

---

## מדריך הפעלה מהירה

הפרויקט מוגדר עם שרת מקומי מבוסס Node.js המאפשר שמירה ישירה של הקואורדינטות לדיסק.

### שלב 1: התקנת תלויות
פתח את מסוף הפקודות (Terminal) בתיקיית הפרויקט והרצ את הפקודה הבאה כדי להתקין את Express:
```bash
npm install
```
*(אם מותקן אצלך PowerShell וחסומות בו הרצות סקריפטים, השתמש ב-`npm.cmd install`)*

### שלב 2: הרצת השרת המקומי
הרצ את הפקודה הבאה כדי להפעיל את שרת הפיתוח:
```bash
npm start
```
השרת יופעל בכתובת: **`http://localhost:3000`**

### שלב 3: כניסה למפה
פתח את הדפדפן והיכנס לכתובת [http://localhost:3000](http://localhost:3000).

## כיצד להשתמש בכלי המיפוי האינטראקטיבי (Editor Mode)

מאחר והמפה גדולה מאוד (7K פיקסלים) ואין לנו מיקומים מוגדרים מראש בקואורדינטות, יצרנו עבורך כלי מיפוי חווייתי וקל:

1. היכנס לאתר ולחץ על כפתור **"מצב עריכה" (Editor Mode)** בסרגל העליון.
2. בסרגל הצד יופיעו שני חלקים: "מקומות שמופו" (כרגע ריק) ו-"מקומות להצבה במפה" (המכיל רשימת אתרים מוכנה מראש ללא מיקום).
3. לחץ על כפתור **"מקם על המפה"** ליד אחד המקומות (לדוגמה: *טאר ואלון*).
4. כעת, סמן העכבר על המפה ישתנה לכוונת (Crosshair). לחץ על המקום המתאים במפה הויזואלית הגדולה שבו מופיע השם "Tar Valon".
5. המערכת תמקם את הסיכה באופן מיידי, תציג הודעה חיובית, ותשמור את הקואורדינטות **ישירות** לתוך קובץ הנתונים `locations.json` בדיסק המקומי שלך!
6. תוכל גם ללחוץ על כל נקודה ריקה במפה כדי לפתוח חלונית **הוספת מקום חדש** מאפס, להזין שמות ותיאורים בשתי השפות, ולשמור!
7. כדי למחוק או לערוך מקומות קיימים, לחץ על המרקר במפה ולחץ על "עריכה" או "מחיקה" (הכפתורים מופיעים רק כאשר מצב העריכה פעיל).
